How to use Text Similarity system.

Pre
1. User must have computer with integrated development environment(ide) or software that can run python program 
	like Spyder, Visual Studio Code etc.
2. User must install python and the libraries (nltk and tkinter) in computer.
3. Change the path on line 62 in TextSimilarity.py to your own path where the file stored and click RUN.

Post
1. Browse the files that you want to check similarity by click the "Browse File" button and click "submit" button.
2. The percentage of similarity will display at the "Similarity Percentage".
